# Carrousel Logos

Ce projet contient un carrousel de logos animé utilisant la bibliothèque **Splide.js**.

## Fonctionnalités
- Défilement automatique des logos.
- Design responsive.
- Personnalisation des paramètres du carrousel (vitesse, espace, etc.).

## Structure des fichiers
- `index.html` : Contient le code HTML principal.
- `images/` : Dossier contenant les images des logos.
- `README.md` : Documentation du projet.

## Comment utiliser
1. Clone ou télécharge ce dépôt.
2. Héberge les fichiers sur un serveur (par exemple, Netlify).
3. Accède au fichier `index.html` pour voir le carrousel.

## Technologies utilisées
- **HTML**
- **CSS**
- **JavaScript**
- [Splide.js](https://splidejs.com/) pour le carrousel.
